# Raspberry Pi Cluster 
This repo describes an architecture for a clustered container platform
using Raspberry Pis. It is very much a work in progress. 

## Implemenetation
Contains a readme and scripts for building the iteration0 version of the cluster on 3 raspberry pi 3Bs


## rpi_cluster
Contains artifacts for Raspberry Pi Docker clustering project

### .archimate files
These files are [Archi](http://www.archimatetool.com) files; you will need Archi to view them. 

[Archimate github](https://github.com/archimatetool)

### views folder
Contains rendered .svg views
